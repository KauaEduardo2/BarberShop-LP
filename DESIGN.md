# Coast Barber Shop - Landing Page

Bem-vindo à documentação oficial da Landing Page (LP) da **Coast Barber Shop**. Este projeto é uma página de captura responsiva, moderna e otimizada para SEO, voltada para uma barbearia de alto padrão.

##  Visão Geral
A Landing Page foi desenhada para atrair, informar e converter clientes em potencial. O design foca em imagens de alta qualidade, contraste com tipografia legível e fácil acesso ao canal de agendamentos.

##  Funcionalidades Principais
- **Otimização de SEO**: Tags meta como Title, Description, Keywords e Open Graph já configuradas para a atração local.
- **Design Responsivo**: Totalmente adaptável a dispositivos Mobile, Tablet e Desktop com o uso e suporte do framework Tailwind CSS.
- **Integração com Google Maps**: Exibição da localização exata do estabelecimento (*333 Marigold Ave, Corona Del Mar, CA 92625, Estados Unidos*) por meio de uma integração interativa via Google Maps IFrame.
- **Interface UI/UX de Alto Padrão**: Uso das tipografias modernas *Space Grotesk* e *Manrope* e estética de paleta de cores harmoniosamente focada no minimalismo luxuoso.
- **Estruturação Semântica**: Código HTML escrito e estruturado de forma legível e semântica (`<nav>`, `<main>`, `<section>`, `<footer>`).

## 🛠 Tecnologias Utilizadas
- **HTML5**: Semântica de ponta para marcação limpa.
- **Tailwind CSS**: Estilização ágil via classes utilitárias importadas com o script CDN.
- **Google Fonts & Material Icons**: Recursos de fontes nativas renderizadas de modo assíncrono.

##  Como rodar o projeto
1. Faça o clone do repositório: `git clone https://github.com/KauaEduardo2/BarberShop-LP.git`
2. Navegue até a pasta do projeto.
3. Abra o arquivo principal `code.html` diretamente em um navegador web de sua preferência (Google Chrome, Edge, Firefox, etc.).
*Não é exigida nenhuma configuração de compilador ou build. O Tailwind é executado dinamicamente via CDN localmente.*

##  Estrutura e Seções
1. **Nav Bar:** Cabeçalho simples e elegante (Glassmorphism).
2. **Hero Section:** Contém o nome da marca, subtítulo agressivo sobre a proposta de valor e chamadas (Call to Action).
3. **Gallery/Sobre:** História sobre o diferencial da barbearia.
4. **Serviços:** Grid inspirado em diagramação de revistas para listagem de preços e categorias.
5. **Contato e Maps:** Área para envio de e-mails, com o mapa de Marigold Ave totalmente integrado, juntamente do expediente.
